# Dreamscape Maze - Frontend

Immersive 3D Multiplayer Horror Maze Game - Client Application

## Tech Stack

- **Three.js** - 3D rendering
- **Cannon-es** - Physics engine
- **Vite** - Build tool
- **PeerJS** - Multiplayer connections

## Setup

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
```

## Features

- First-person horror maze exploration
- 10 different trap types with visual effects
- Ghost AI with pathfinding
- Multiplayer support
- Mobile touch controls
- PWA support
